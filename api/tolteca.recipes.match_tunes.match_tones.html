match_tones¶

tolteca.recipes.match_tunes.match_tones(left, right, eps=2000.0, shift_from_right=0.0, match_col='fr', join_type='inner')[source]¶

Return a table with tones matched.

This function makes use the stilts utility.

Parameters
left: astropy.Table

The left model params table.

right: astropy.Table

The right model params table.

eps: float

The error to tolerate in Hz.

shift_from_right: float

The frequency shift to apply to the right.

match_col: str

The column to use for the match. Default is the resonance frequency fr.

join_type: str

Join type to use for the output table.
