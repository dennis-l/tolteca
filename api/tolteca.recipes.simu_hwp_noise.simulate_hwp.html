simulate_hwp¶

tolteca.recipes.simu_hwp_noise.simulate_hwp(Qr, readout_noise, hwp_var_temp)[source]¶

Make simulated timestream and calculate some useful quantities of the HWP under some assumed parameters.

Parameters

  • Qr (float) – Qr of the detector.

  • readout_noise (float) – Readout noise specified in the same unit as Qr.

  • hwp_var_temp (float) – The HWP temperature variation, peak-to-peak.

Returns

Dict that contains all stated local variables.

Return type

dict
