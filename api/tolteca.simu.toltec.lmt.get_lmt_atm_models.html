get_lmt_atm_models¶

tolteca.simu.toltec.lmt.get_lmt_atm_models(name, *args, **kwargs)[source]¶

Return the LMT atmosphere model and the transmission model.

Parameters
name{“am_q25”, “am_q50”, “am_q75”}, default=”am_q50”

The source dataset used in the model. The “am_q*” data are constructed with the am code https://www.cfa.harvard.edu/~spaine/am/, with quartiles set to 25 and 50, respectively.

*args, **kwargs

Arguments passed to the _Model constructor.
